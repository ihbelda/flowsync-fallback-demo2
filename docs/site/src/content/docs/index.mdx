## ¿Qué es FlowSync?

FlowSync es un starter kit full-stack con autenticación por **access tokens**, repo
hilo conductor del máster **AI4Devs** (LIDR). Este sitio centraliza su documentación:
guías de arranque, referencia de la API y las decisiones de arquitectura (ADRs).
